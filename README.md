OmniTrack: Full-Stack Tracking System
A robust tracking application featuring a FastAPI backend and a React frontend, optimized for performance and scalability. This project demonstrates my transition from ECE concepts to high-level software development with a focus on Python-based backend architecture.

🚀 Tech Stack
Backend: FastAPI (Python)

Frontend: React.js, Tailwind CSS

Database: SQLAlchemy

State Management: Hooks & Context API

✨ Key Features
Asynchronous API: Built with FastAPI for high-performance request handling.

Relational Mapping: Database schema managed via SQLAlchemy/PostgreSQL.

Responsive UI: Clean, modern frontend designed with React and Tailwind.

CRUD Operations: Full Create, Read, Update, and Delete functionality for tracking data.

🛠️ Installation & Setup
Backend (FastAPI)
Navigate to the /backend directory.

Create a virtual environment: python -m venv venv.

Install dependencies: pip install -r requirements.txt.

Start the server: uvicorn main:app --reload.

Frontend (React)
Navigate to the root directory (where package.json is located).

Install dependencies: npm install.

Start the development server: npm start.
